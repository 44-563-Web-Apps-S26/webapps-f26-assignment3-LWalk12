  3. Lay out the runners and instructions
  4. Seperate the pieces from the runners
    • Using your cutting tool cut the gates attaching the parts to the kit
    • Using the same tool carefully remove the nibs off the piece
    • OPTIONAL: you can use a small file to smooth and clean the pieces
  5. Using the instructions, assemble the parts
